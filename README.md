## Installation

	1. Install "SSH Toggle and Port" from Cydia / Sileo. 
	2. In Settings -> OpenSSH -> change SSH port to "22,2222" (Without quotations)
	3. Run the shortcut.

	* If it's not working, change it to "22,22222" in OpenSSH toggle and also "22222" in shortcut.
